# Digital Heroes Shopify Theme Project

This repository now contains a functional Shopify Online Store 2.0 starter theme implementation for the assignment.

## Included requirements
- 3-color palette foundation (primary/secondary/accent)
- Homepage sections:
  - Hero banner + CTA
  - Featured collections
  - Best-selling products
  - Benefits / USP
  - Testimonials
  - Trust badges
  - Newsletter signup
- Header + footer navigation scaffolding
- Dedicated templates for About, Contact, FAQ, and Shipping & Returns pages

## Files to upload to Shopify
Upload this repository as a custom theme ZIP (or push with Shopify CLI):
- `layout/`
- `templates/`
- `sections/`
- `assets/`
- `config/`
- `locales/`

## Post-upload steps in Shopify admin
1. Assign collections in **Featured collections** and **Best sellers** sections.
2. Create menus:
   - `main-menu`
   - `footer`
3. Create pages and assign templates:
   - `page.about`
   - `page.contact`
   - `page.faq`
   - `page.shipping-returns`
4. Install required apps from assignment and position blocks where needed.
